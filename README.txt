Duel Game v6.15 - Update

New features:
- Admin can deal 2 random action cards to all players (button in admin panel)
- One-time cards are removed from hand after use
- Level up system: every 100 XP = +1 level, +10 max HP, restore 10 HP
- Duels with hit locations (head 50, torso 40, arm/leg 30)
- Evasion chance: 30% at level 1, +5% per level (max 95%)
- XP rewards: winner +40, loser +20, draw both +20

Run:
1) npm install
2) npm start
3) Open http://localhost:3000

All old features preserved (healer, bartender, dealer, avatars, roles).
